# Sword Circle Pen — Project Constitution

> “This is not a website.  
> It is a cinematic digital archive of a life devoted to service, intellect, literature, science, and human transformation.”

---

# Core Philosophy

Sword Circle Pen must feel:
- cinematic
- contemplative
- atmospheric
- literary
- intelligent
- timeless
- emotionally restrained

This project is NOT:
- a traditional portfolio
- a standard author website
- a SaaS-style experience
- a generic content platform

The website should feel like:
- entering memory
- discovering journals
- walking through intellectual history
- uncovering fragments of a life lived deeply

Every design and engineering decision must support:
- emotional pacing
- immersive storytelling
- intellectual depth
- human authenticity
- quiet elegance

---

# Experience Direction

The experience should unfold as:
- scenes
- emotional transitions
- environmental storytelling
- cinematic scroll sequences

NOT:
- sections
- grids
- dashboards
- boxed layouts

Scrolling should feel like:
- drifting through memory
- opening archival material
- ascending through thought

---

# Emotional Themes

The project explores:
- military service
- identity
- migration
- scholarship
- environmental responsibility
- literature
- scientific thought
- veteran reintegration
- transformation through education

---

# Visual Language

## Allowed

- atmospheric gradients
- grain textures
- fog
- layered parallax
- editorial typography
- slow cinematic motion
- archival imagery
- subtle particles
- negative space
- manuscript aesthetics

## Forbidden

- loud gradients
- startup aesthetics
- excessive glassmorphism
- dashboard UI
- overly bright palettes
- trendy social-media design
- excessive animation
- cluttered interfaces
- generic website sections

---

# Typography Philosophy

Typography is a storytelling medium.

Typography should feel:
- literary
- editorial
- poetic
- cinematic
- intelligent

Use:
- elegant serif headlines
- restrained sans-serif body text
- asymmetrical layouts
- large quote fragments
- staggered reveals

Avoid:
- overly modern startup fonts
- dense text walls
- excessive UI labeling

---

# Motion Philosophy

Motion should feel like:
- memory
- wind
- drifting paper
- breath
- contemplation

Avoid:
- bounce animations
- hyperactive transitions
- excessive scroll gimmicks
- motion without narrative purpose

---

# Technology Stack

## Frontend

Preferred:
- Next.js App Router
- TypeScript
- TailwindCSS
- GSAP
- ScrollTrigger
- Lenis
- Framer Motion

Optional:
- Three.js
- React Three Fiber
- shadcn/ui
- 21st.dev components
- Aceternity UI
- Magic UI

---

# Frontend Architecture

Build:
- systems
- scene engines
- reusable motion primitives

NOT:
- static page sections

Preferred architecture:

```txt
SceneRenderer
 ├── HeroScene
 ├── TimelineScene
 ├── ArchiveScene
 ├── VeteranMissionScene
 ├── SpeakingScene
 └── HorizonScene
```

---

# Component Philosophy

Good:
```txt
ParallaxLayer
AtmosphericCanvas
FloatingManuscript
NarrativeQuote
ArchiveReveal
EnvironmentalFade
```

Bad:
```txt
AboutSection
BlogGrid
ServicesCards
TeamSection
```

---

# WordPress Philosophy

WordPress is the:
- archive engine
- editorial system
- publishing backend
- media library
- content management layer

WordPress is NOT responsible for:
- presentation
- frontend rendering
- animation
- storytelling

---

# Headless CMS Architecture

Use:
- Headless WordPress
- WPGraphQL
- Advanced Custom Fields (ACF)

Nestor should only interact with:
- WordPress Admin

The frontend consumes:
- structured content
- custom post types
- metadata
- media assets
- timeline data

---

# Custom WordPress Plugin

A custom WordPress plugin should manage:
- content models
- API exposure
- schema definitions
- custom fields
- future extensibility

---

# Custom Post Types

```txt
scp_essay
scp_book
scp_archive
scp_speaking
scp_veteran_resource
scp_quote
scp_field_note
scp_timeline_event
```

---

# Taxonomies

```txt
theme
discipline
location
institution
environment
literary_style
war_era
```

---

# API Layer

Preferred:
- WPGraphQL

Avoid tightly coupling frontend logic to:
- WordPress themes
- page builders
- traditional templates

Frontend should consume:
- structured scene data
- narrative fragments
- archive materials
- timeline events
- speaking content
- veteran resources

---

# Extensibility Philosophy

The system must support future additions without redesigning the core architecture.

Potential future modules:
- AI narration
- speaking portal
- veteran mentorship system
- documentary archive
- environmental visualizations
- interactive maps
- podcast integration
- educational resources
- university partnerships

All future systems should plug into:
- existing scene architecture
- WordPress data models
- reusable rendering systems

---

# Performance Rules

Cinematic does NOT mean slow.

Requirements:
- lazy load scenes
- optimize assets
- limit particle systems
- avoid excessive WebGL
- optimize typography loading
- use server components where appropriate
- maintain accessibility

---

# AI Agent Rules

AI coding agents must:
- preserve cinematic tone
- maintain emotional pacing
- use semantic HTML
- prioritize accessibility
- support modularity
- preserve atmospheric restraint

AI agents must NOT:
- inject SaaS UI patterns
- flatten storytelling
- over-modernize typography
- convert narrative scenes into cards/grids
- introduce unnecessary complexity

---

# Narrative Structure

The experience should loosely follow:

1. The Ascent
2. The Man
3. The Archive
4. Earth Day Eulogy
5. Veteran Transfer Project
6. Public Speaking
7. The Philosophy
8. The Horizon

These are narrative chapters, not webpage sections.

---

# Final Creative Directive

Sword Circle Pen exists to communicate:
- a life of service
- intellectual resilience
- environmental consciousness
- transformation through education
- the relationship between science and humanity
- the emotional aftermath of war
- the dignity of thoughtful work

Visitors should leave feeling:
- contemplative
- emotionally moved
- intellectually curious
- inspired toward deeper thought

The experience should feel remembered long after the visitor leaves.
